Jelly

pbjelly.exe can generate a local HTML report after testing safety checks and request latency.

Run:

.\build\pbjelly\pbjelly.exe 127.0.0.1 8083 build\public\jelly-report.html

For jelly builds in build\pbjelly, write the report to build\pbjelly\public\jelly-report.html.

Open the latest generated report
