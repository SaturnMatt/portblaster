Validation tools

The pbjelly folder contains validation builds pbj20.exe, pbj50.exe, pbj100.exe, plus pbjelly.exe. Jelly probes normal pages, rejected paths, malformed request lines, unsupported methods, bind mode, queue saturation, defense counters, and load latency with average, p50, p95, and max timing.

Notes on size costs

Windows PE output moves in coarse chunks, so costs are practical measurements rather than perfect source-code weights. Features can share imports and helpers, so isolated and marginal costs differ.
