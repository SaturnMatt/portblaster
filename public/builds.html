Builds

The project keeps two useful binaries:

  • portblaster.exe: smallest release build.
  • portblaster-check.exe: debug-friendly build for validation and analysis.
  • pbprobe.exe: separate safety and performance probe.

The probe checks normal pages, rejected paths, malformed request lines, unsupported methods, and repeated request latency.

The check build accepts --start for automated local validation. The tiny release build stays manual.
